# Teslo API

1. Clonar el repositorio
2. Ejecutar
```
npm install
```
3. Tener Nest CLI intalado
```
npm i -g @nestjs/cli
```
4. Clonar el archivo ```.env.template``` y renombrar la copia a ```.env```

5. Levantar base de datos
```
docker-compose up -d
```

6. Levantar app: ```npm run start:dev```

7. Ejecutar SEED
```
http://localhost:3000/api/seed
```
